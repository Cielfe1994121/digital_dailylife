# Digital Daily Life (The Art of Code & Sensation) 🖱️✨

**日常に潜む「物理的な快感」をコードで再構築するプロジェクト (進行中)**

## 📖 概要 (Overview)
**「触り心地（Tactile）」をデジタルに翻訳する。**

私たちは日々、無意識のうちに物理的な「快感」を感じています。
梱包材を潰す時の抵抗感、磁石が吸い付く瞬間の加速、結露した窓ガラスを拭う指の感触。

本プロジェクト **"Digital Daily Life"** は、そうした日常の些細なインタラクションを、物理演算とプロシージャルサウンドを用いてデジタル空間上に再現する試みです。
単なるアニメーションではなく、マウス操作を通じた **「重み」「抵抗」「開放感」** の再現に主眼を置いています。

## 目的
日常の物理現象には、数式で記述できる「美しさ」と「快感」が潜んでいる。

例えば、磁石がくっつく瞬間の加速度（逆二乗則）や、吸盤が剥がれる瞬間の空気の振動。これらをコードに落とし込み、視覚と聴覚を通じて「疑似的な触覚」として脳に届けること。

そう思い至り、本プロジェクトは始動した。

そこから、「日常の物理をハックする」というコンセプトのもと、以下のツール群を実装した。

## 📂 収録作品 (Works)

### 1. Rain (窓の結露と再生)
雨の日の曇った窓ガラスを指で拭う行為の再現。
* **Wipe & Regen:** マウスで拭うと結露が消え、外の景色がクリアになる。しかし、放置するとじわじわと白く曇りが再生していく（`Alpha`値の加算シミュレーション）。
* **Atmosphere:** 物理演算による雨粒の垂れと、ぼかされた夜景の描画による環境音的な視覚表現。

### 2. The Magnetic Snap (磁石の吸着)
「カチッ」とハマる、あの瞬間の快感。
* **Gapless Contact:** 独自の矩形衝突判定（AABB）とサブステッピング処理により、隙間ゼロでの完全吸着を実現。
* **Physics:** 距離の2乗に反比例する引力の実装。遠くでは穏やかに、近づくと急激に加速して衝突する。
* **Interaction:** `R` / `L` キーによる90度回転、右クリックによる「強制引き剥がし」など、デジタルならではの磁石遊び。

### 3. The Bubble Wrap (無限プチプチ・改)
ただクリックするだけではない、「溜め（Tension）」と「弾け（Release）」の美学。
* **Pressure Simulation:** マウスの押し込み時間に応じて気泡が歪み、色が変わり、限界圧力を超えるとパーティクルが弾け飛ぶ。
* **Visual Feedback:** 圧力によるハイライトの歪みや、破裂直前の微振動（シェイク）による「耐える」演出。

### 4. The Suction Cup (吸盤と音響)
ゴムの粘り気と、剥がれる瞬間の爽快感。
* **Audio-Visual Sync:** コードで生成されたプロシージャルサウンド（波形生成）と物理演算が同期。引っ張る距離に応じて「きゅ〜〜っ」という音のピッチや音量が変化する。
* **Vacuum Physics:** 引っ張り続けると真空度が低下し、限界を迎えると「スポッ！」と剥がれ、バネの反動でマウス位置に戻ってくる。

## 📐 技術的アプローチ (Technical Approach)

### 1. Physics & Haptics（物理と触覚）
視覚だけでなく「手ごたえ」を感じさせるための工夫。

* **Sub-stepping (サブステッピング):**
    磁石の吸着において、1フレームの間に物理計算を10回行うことで、高速で衝突しても「めり込み」や「すり抜け」を防ぎ、カチッとした硬い物体の質感を表現。
* **Tension & Release (緊張と緩和):**
    吸盤やプチプチにおいて、ユーザーの入力（ドラッグ/長押し）に対して「抵抗値（Resistance）」を返し、閾値を超えた瞬間に一気に「開放（Release）」させることで、脳に快感を与える。

### 2. Procedural Sound（手続き型音響）
フリー素材を使わず、物理現象に完全に同期した音を生成。

* **Waveform Synthesis:**
    `Sounds/make_sounds.py` にて、サイン波やノイズを数学的に合成。「きゅー（摩擦音）」や「スポッ（急激なピッチダウン）」をバイナリとして生成し、物理イベントと1/60秒単位で同期させる。

### 3. Visual Feedback（視覚フィードバック）
* **Alpha Blending:**
    結露の「徐々に白くなる」表現や、吸盤の「真空ゲージ」の可視化。
* **Deformation:**
    圧力がかかった際のゴムやビニールの歪みを、円の描画パラメータを動的に変形させることで表現。
